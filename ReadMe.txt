Nama website kami adalah World E-News

Konfigurasi awal ketika sudah download folder dari repository github kami adalah :
1. Install vue terlebih dahulu dengan masukan perintah " npm install -g @vue/cli "
2. Setelah itu buka folder project vue dan kemudian masuk ke terminal.
3. Setelah itu masukan perintah " npm update " untuk update package yang kami gunakan.
4. Setelah itu jalankan perintah " npm run serve " untuk menjalankan website ini.
5. jika sudah silahkan akses website sesuai dengan port yang ada di terminal.

Note : 
- Project kami tidak bisa di hosting karena api yang kami gunakan tidak allow untuk di hosting sehingga ketika di hosting api tidak memberikan data karena tidak diizinkan. Tetapi untuk hostingannya sudah ada dan sudah bisa diakses, namun datanya tidak bisa.
